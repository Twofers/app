Restricted access

Admin Login

Admin access is limited to active users in the server-side admin allowlist.

Authentication Boundary

Use Supabase Auth to create a session, then store a short-lived access token in this browser as twofer_admin_access_token. Server functions still verify the authenticated user against admin_users.

Do not paste service-role keys, Stripe keys, API keys, or production secrets into this page.

Security Checks

Every admin action must independently check authentication, active admin role, permission, valid target state, and audit logging. The UI is only a convenience layer.
