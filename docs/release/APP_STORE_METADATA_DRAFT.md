## Promotional Text

## Description

Twofer helps people discover live local offers from nearby businesses. Browse local specials, claim an offer, save favorite businesses, and redeem in person with a QR or staff verification flow.

For participating businesses, Twofer provides reviewed merchant tools for creating controlled, time-limited local offers during slower hours. Merchant accounts are reviewed before activation.

Twofer is launching first in select Dallas neighborhoods.
